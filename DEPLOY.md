# Deploy SAS Connect Website

Code is pushed to: https://github.com/nocodebrain/sas-connect-website

## Deploy to Railway (2 minutes):

1. Go to https://railway.app
2. Click "New Project"
3. Select "Deploy from GitHub repo"
4. Choose: `nocodebrain/sas-connect-website`
5. Railway auto-detects Next.js
6. Click "Deploy Now"
7. You'll get a URL like: `sas-connect-website-production.up.railway.app`

## Deploy to Vercel (2 minutes):

1. Go to https://vercel.com
2. Click "Add New Project"  
3. Import: `nocodebrain/sas-connect-website`
4. Click "Deploy"
5. You'll get a URL like: `sas-connect-website.vercel.app`

## Local Development:

```bash
npm install
npm run dev
# Open http://localhost:3000
```
